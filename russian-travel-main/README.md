# Путешествия по России

## Описание проекта

Одностраничный адаптивный сайт о внутреннем туризме в России. Страница рассказывает о пяти уникальных местах страны: Куршской косе, Кольском полуострове, Алтае, зимнем Байкале и Карелии. Проект создан в рамках учебной программы Яндекс Практикум.

Ссылка на проект: [https://username.github.io/russian-travel/](https://username.github.io/russian-travel/)
*(замените username на своё имя пользователя GitHub)*

### Функциональность

- Адаптивная вёрстка для 5 брейкпоинтов: 1280px, 1024px, 768px, 425px и 320px
- Отсутствие горизонтального скролла на любой ширине от 320px до бесконечности
- Анимированный заголовок на секции Lead с изображением купе поезда
- Секция Intro с двухколоночным макетом: заголовок слева, текст и статистика справа
- Фотогерид из 9 снимков с адаптивной сеткой
- Пять секций достопримечательностей с двухколоночным макетом: заголовок и ссылка слева, фото и текст справа
- Секция Cover с фоновым изображением и интерактивной ссылкой
- Плавное изменение прозрачности ссылок при наведении

### Технологии

- **HTML5** — семантическая разметка (`header`, `main`, `footer`, `section`, `article`, `figure`, `figcaption`, `nav`, `blockquote`)
- **CSS3** — Flexbox, CSS Grid, медиазапросы (`@media`), свойство `transition`, `object-fit`, `aspect-ratio`
- **БЭМ (Nested)** — методология именования классов и файловая структура `blocks/`
- **Адаптивность** — пять брейкпоинтов: ≥1280px, ≤1279px, ≤1023px, ≤767px, ≤424px
- **normalize.css** — сброс стилей браузера
- **Inter** — шрифт, подключённый через Google Fonts
